# PARA Memory System

Workspace durable knowledge root: `para/`

## Structure
- `para/projects/`
- `para/areas/`
- `para/resources/`
- `para/archives/`

## Rules
- Durable knowledge belongs here.
- Prefer structured summaries over raw transcripts.
- Supersede outdated facts instead of silently deleting history.
- Never store secrets.
