# Design Tokens and UX Rules

## Color / semantics
- status.todo = neutral
- status.in_progress = cool active
- status.blocked = amber warning
- status.done = subdued success
- priority.high = warm emphasis
- priority.critical = strongest emphasis

## Layout
- desktop grid: 12 columns
- max width: 1440–1600px
- outer padding: 32–40px
- panel gaps: 20–24px
- card padding: 20–24px

## Board ratios
- Next up: 1.15
- Blocked: 0.85
- Recently changed: 1.0

## Copy rules
Use:
- Live • updated 18:16
- No active blockers
- Due in 3 days
- Updated 14m ago
- Source delayed

Avoid vague filler language.
